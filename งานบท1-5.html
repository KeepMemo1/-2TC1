    การขึ้นบรรทัดใหม่(Paragraph,P)

P เป็นคำสั่งให้ขอความขึ้นบรรทัดใหม่และเว้น 1 บรรทัด /P

P   /P เป็นคำสั่งให้ขอความขึ้นบรรทัดใหม่และเว้นหลายบรรทัด
    การจัดตำแหน่งข้อความ(Align)

การจัดตำแหน่งสามารให้แท็กPหรือDiv

ข้อแตกต่างแท็ก P จะเว้นระยะบรรทัดห่างกว่า Div

ตัวอย่างการเขียน

(P Align = left) จัดตำแหน่งข้อความชิดซ้าย

(Div Align = cneter ) จัดตำแหน่งข้อความ กึ่งกลาง

การย่อหน้าและการเยื้อง

การย่อหน้าใช้แท็ก DD

การเยื้องใช้แท็ก Blockquote

ตัวอย่างการเขียน

(Blockquote) จัดให้ข้อความแต่ละบรรทัดเยื้องด้านหน้าเท่ากันทุกบรรทัด แต่ยังไม่ได้จัดให้บรรทัดแรกมีการย่อหน้า (/Blockquote)

(Blockquote) (DD) จัดให้ข้อความแต่ละบรรทัดเยื้องด้านหน้าเท่ากันทุกบรรทัด และมีการย่อหน้าในบรรทัดแรกโดยใช้แท็ก DD (/DD)(/Blockquote)

(hr size = หน่วยเป็น pixet ) กำหนดความหนาของเส้น

(hr width = หน่วยเป็น%,pixet ) กำหนดความยาวของเส้น

ตัวอย่างการเขียน

(hr width=150% size=20px) ความยาว150% ความหนา20px

การตกแต่งขอความด้วยแท็ก(Font)

ใช้กำหนดรูปแบบตัวอักษรและข้อความ

(font color = red) กำหนดสีตัวอักษร

(font size= 5) กำหนดขนาดตัวอักษรได้ตั้งแต่1-7

(font face = Angsana new ) กำหนดรปแบบอักษร

การตกแต่งขอความด้วยแท็ก(Basefont)

ใช้กำหนดรูปแบบตัวอักษรและข้อความภายใต้แท็ก(Basefont) เป็นแท็กเดี่ยว

ตัวอย่างการเขียน

(Basefont color = red size = 5 face = Angsana new ) ตัวอักษรเป็นสีแดง ขนาดตัวอักษรเท่ากับ5 รูปแบบอักษรเป็นแบบ ANGSANA NEW
